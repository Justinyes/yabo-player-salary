# 津门鸭脖球员薪资软件 V3

一个用于美职篮全明星街头 3V3 赛事的纯前端球员薪资管理工具。打开 `index.html` 即可使用，不需要安装服务器或数据库。

## 功能

- 球员价格库查询、筛选和加入阵容
- 3V3 阵容工资帽校验
- 工资帽随淘汰赛轮次成长参考
- 保存队伍到浏览器本地存储
- 管理员登录后调整比赛薪资
- 管理员登录后导入完整名单、合并新球员、恢复默认价格库
- 导出当前价格库 JSON
- 所有访客自动读取 GitHub 上发布的 `players.json`

## 使用方式

直接双击打开 `index.html`。

默认管理员密码在 `script.js` 顶部：

```js
const ADMIN_PASSWORD = 'admin123';
```

需要改密码时，修改这里即可。

## 重要说明

网站上的统一公开名单来自 `players.json`。管理员在网页里做的调整是临时修改，需要点击“导出 JSON”，再将导出的文件改名为 `players.json` 并上传覆盖 GitHub 仓库中的同名文件，其他用户才能看到。

这是纯静态网页，管理员登录只适合本地赛事管理时防误操作。代码开源后，任何人都能看到并修改前端密码。如果你后续需要真正防止别人改价格，需要加后端登录、数据库和权限校验。

## 开源发布步骤

1. 新建 GitHub 仓库，比如 `yabo-player-salary`.
2. 把本文件夹里的 `index.html`、`style.css`、`script.js`、`players.json`、`README.md`、`LICENSE` 上传到仓库。
3. 仓库设置里开启 GitHub Pages，Source 选择默认分支，目录选择根目录。
4. 等待 GitHub Pages 生成网址后，把网址发给参赛者使用。
5. 如果要让别人参与改进，建议用 Pull Request 合并修改。
